### Load the Extension (30 seconds)

1. Open Chrome: `chrome://extensions/`
2. Enable "Developer mode" (toggle in top-right)
3. Click "Load unpacked"
4. Select the `/extension` folder
5. Done!

### Use It (3 steps)

1. Visit any website with images
2. Drag an image toward the bottom of the screen
3. Drop it on the beautiful floating dock

**That's it!** Your image is downloaded with an organized filename.
